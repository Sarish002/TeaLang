# TeaLang
My own programming language!

# Usage
* Upload *Program.tea*
* Write your code
* Run *main.py*

# Setting and changing variables
To set variables:
`set (var) to (val)`

To change variables:
`change (var) to (val)`

To give input:
`set (var) to steal | (prompt) |`

# Print
To print:
`show (var)`

# If - Else
To write conditional blocks:

`only (cond): (comm)`

`otherwise: (comm)`

# *Python Script*
To write *Python* in TeaLang:
`Python >>>
(Your code)
<<<`
